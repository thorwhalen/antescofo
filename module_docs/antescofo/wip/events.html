antescofo.wip.events

Event representation for score playback.

This module defines the Event data structure used by the SimpleScorePlayer.

class antescofo.wip.events.Event(type: str, duration: float, data: str, actions: list[str] = <factory>)[source]

A musical event in a score.

Represents a single event (note, chord, etc.) with associated actions.

type

Event type (e.g., “NOTE”, “CHORD”)

Type:

str

duration

Duration in beats

Type:

float

data

Event-specific data (e.g., “C4 60” for MIDI note)

Type:

str

actions

List of actions to execute when event triggers

Type:

list[str]

Example

>>> event = Event("NOTE", 1.0, "C4 60", ['print "Hello"'])
>>> event.type
'NOTE'
>>> event.duration
1.0
